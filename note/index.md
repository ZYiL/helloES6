#   ES6(ES2015)
####    兼容性
-   IE10+
-   Chrome
-   FireFox
-   移动端
-   NodeJs

####    兼容IE10以下的解决方案
-   在移动端或者混合开发中多用ES6，老版本中不用ES6
-   编译、转换
    -   在线转换，页面表现卡顿
    -   提前编译（更推荐）  工具Babel===browser.js
